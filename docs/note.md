# 创建 package
```
flutter create --template=package <your_package_name>
```
这里的 ‘your package name’ 需要使用下划线

# 修改 pubspec.yaml 元信息

  1. description - 这个package的功能描述
  2. author - 作者； 
    
    ```
    单个作者：
    author: Rain.xia <406330691@qq.com>
    ```
    ```
    当有多个作者时：
    authors:
      - Rain.xia <406330691@qq.com>
      - Raining.xia <xxxx@qq.com>
    ```
